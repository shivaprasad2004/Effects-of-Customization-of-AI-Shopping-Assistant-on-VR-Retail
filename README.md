## 🚀 Core Overview
This project studies user behavior in 3D retail using React Three Fiber, Node.js, and advanced AI models. It tracks how AI personalization and blockchain trust markers affect user confidence and conversion.

## 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Vite, TailwindCSS, Redux Toolkit, Framer Motion, Three.js (R3F)
- **Backend**: Node.js (Express), Socket.io, Mongoose (MongoDB), PostgreSQL, Redis
- **AI Service**: GPT-4o, TensorFlow, OpenCV (Affective AI), Python FastAPI
- **Blockchain**: Solidity (Hardhat), Ethers.js, IPFS, MetaMask, Hyperledger Fabric Simulation
- **Infrastructure**: Docker, Nginx, AWS S3

## 🔑 Key Features
### 1. 3D VR Store
- Interactive first-person shopping with a **Virtual Trial Room**.
- Real-time navigation via AI chatbot commands: `[NAVIGATE:ZONE]`.
- 360° product exploration and spatial dwell-time tracking.

### 2. AI & Emotion Engine
- **Emotion Recognition**: Predicts user sentiment to trigger dynamic discounts and token rewards.
- **ShopBot AI**: Conversational concierge powered by GPT-4o with real-time intent detection.
- **Hybrid Recommendations**: Collaborative + Content-based filtering using behavioral dwell-time data.

### 3. Blockchain & Loyalty
- **Authenticity Certificates**: Premium products verified via Solidity smart contracts and IPFS hashes.
- **Secure Payments**: Blockchain-enabled escrow payments simulated with MetaMask integration.
- **$SHOP Token**: ERC-20 loyalty system rewarding users for engagement and positive emotions.

## ⚙️ Installation
1. **Clone the repo**:
   ```bash
   git clone https://github.com/shiva/vr-shopping-assistant.git
   cd vr-shopping-assistant
   ```
2. **Setup Env**:
   Update `backend/.env` with your OpenAI API key and MongoDB URI.
3. **Launch Project**:
   ```bash
   # Backend
   cd backend && npm install && npm run dev
   # Frontend
   cd frontend && npm install && npm run dev
   ```
